--JavaScript Snippets:
Como nome sugere, são trechos de códigos utilizados no Visual Studio para agilizar o desenvolvimento de código.

--REST Client:
Faça requests direto do VSCode e veja as respostas de cada requisição.

--Emmet:
Facilita bastante a escrita do HTML, provendo abreviações poderosas, snippets dinâmicos e diversos atalhos para escrever HTM.

--Path Intellisense:
Ajuda de preenchimento de código que inclui inúmeras funcionalidades: Listar Membros, Informações do Parâmetro, Informações Rápidas e Completar Palavra.

--Import Cost:
Exibe o tamanho dos módulos que você está importando para o seu código em JS.
